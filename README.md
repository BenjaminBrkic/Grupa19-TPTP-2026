# Tuzla Stay — TPTP Završni projekat 2025/2026

Web lokacija za pregled i pronalazak smještaja u Tuzli.

## Tema

**Turistički vodič — smještaj u Tuzli**  
Platforma koja omogućava pregled, filtriranje i kontaktiranje u vezi sa smještajem u Tuzlanskom kantonu.

## Stranice

- `index.html` — Glavna stranica: hero sekcija, filteri, kartice smještaja, mapa i aside sekcija
- `sadrzaj.html` — Detaljna stranica: tabela, galerija slika, image map, ugniježđene liste i bookmark navigacija
- `kontakt.html` — Kontakt forma sa JavaScript validacijom i interaktivnim elementima

## Tehnologije

- HTML5 (semantički elementi, `<picture>`, image map, tabela, bookmark navigacija)
- CSS3 (custom properties, Flexbox, Grid, media queries, `@keyframes`, tranzicije)
- Vanilla JavaScript (bez frameworka i biblioteka)

## Pokretanje projekta

Otvoriti `index.html` u web browseru.  
Sve putanje su relativne i projekat je prilagođen za lokalno pokretanje i GitHub Pages.
